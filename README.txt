Технологии:
Приложение написано на java с использованием сервлетов.
Запуск происходит на tomcat сервере, версии 9.0

Цель:
Прилрежние выводит в браузере страничку приветствия.

Запуск:
Нужно из папки с проектом выполнить команду
docker build -t welcomer .
что бы получить имадж приложения
по завершению выполнить
docker run -p 8080:8080 welcomer:latest

Перейти по УРЛ
http://localhost:8080/wellcomer/index.jsp

Enjoy!
